### Iniciando o projeto 

```bash
# Clone este projeto
$ git clone git@github.com:andersonlopeswm/manager-by-microservices.git

# Acessa o projeto
$ cd manager-by-microservices

# Instala as dependencias
$ yarn

# Executa em modo de desenvolvimento com auto reload
$ yarn dev

# build para executavel linux
$ yarn dist:linux

```
